Lesson 1: Introduction

What is HTML?

HyperText Markup Language (HTML) is the language used to write web pages.
HTML elements consist of tags and content, most elements will have an opening and closing tag.

Basic structure - HTML Documents
  • A doctype declaration - the most updated version is HTML5
  • HTML element
  • Head element
  • Body element

Tip: A boilerplate code can be copied from the internet with this structure already pre-made.


What is CSS?

Cascading Style Sheets (CSS) creates the visuals for the web page. This includes:

  • Layout
  • Colour
  • Size
  • Font

CSS can be added to HTML via inline, internal or external styles, the preferred option is
the external style which involves the creation of a seperate .css file.
The separate file is then linked in the head element.

An example of HTML used for a website is the Vision Australia website which has been identified as a WCAG AAA website.
